# Open API Project

## Link to Repository: https://github.com/AndrewDass1/Open-API-Project

This project is titled "Getting Current Weather for a City". This project homepage is the "index.html" file and has the title "Getting Current Weather for a City" and below it, text stating "Enter City Name" in the text box and then press the "Submit" afterward. This project scrapes from the Open-Meteo (https://open-meteo.com/) (1) weather API, where a fetch was implemented in order to first retrieve the city's latitude and longitude, and then another fetch uses those coordinates to scrape the current weather and temperature in Celsius for the city that the user inputs into the text box. After they submit the name of the city and is found in Open Meteo's database, more additional text appears on the HTML page stating the name of the city, a picture of its current weather, the description of the current weather and also the current temperature in Celsius and Farhenheit.
